Welcome to the Introduction to Python Network Programming course. This directory, PythonNetwork, is where you find support files related to the class exercises. It is also where you will be doing your work.

  • The Exercises/ folder is where you find all the class exercises.

  • The Data/ folder is where you find data files, scripts, and other files used by the exercises.

  • The Solutions folder contains complete solution code for various exercises. Each exercise has a dedicated folder which matches the section and problem number.
