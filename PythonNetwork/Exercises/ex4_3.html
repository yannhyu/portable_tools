Exercise 4.3

Objectives:
  • Run your own webserver

Python has a library modules that can be used to run a simple web server. Using the notes as a guide, put the following code into a file webserve.py: 

# webserve.py
# Note: The program must be placed in the PythonNetwork/ folder

from BaseHTTPServer import HTTPServer
from SimpleHTTPServer import SimpleHTTPRequestHandler
import os
os.chdir("Exercises")
serv = HTTPServer(("",8080),SimpleHTTPRequestHandler)
serv.serve_forever()
Try running your server. This server should deliver files from the PythonNetwork/Exercises folder. Test it out by clicking on the following link:
http://localhost:8080/index.html
Congratulations! You're now running your own web server.
