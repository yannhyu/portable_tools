Exercise 4.1

Objectives:
  • Learn about HTTP by manually issuing a request

To learn about the basics of HTTP, we're going to manually issue a request and look at the response.

First, create a Python string that contains a simple HTTP request. Note: this must be typed in exactly for it to work.

>>> request = """\
GET /download/ HTTP/1.0
Host: www.python.org

"""
>>>

Now, open a socket connection to a remote web server, send the above request, and look at the response:

>>> from socket import *
>>> s = socket(AF_INET, SOCK_STREAM)
>>> s.connect(("www.python.org",80))
>>> s.sendall(request)
46
>>> data = s.recv(8192)
>>> print data
... look at the output ...

When looking at the response, carefully study the response line, headers, and content.
